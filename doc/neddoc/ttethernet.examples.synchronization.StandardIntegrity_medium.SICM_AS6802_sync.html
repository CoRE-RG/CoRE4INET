Network SICM_AS6802_sync

Package: ttethernet.examples.synchronization.StandardIntegrity_medium
File: examples/synchronization/StandardIntegrity_medium/AS6802_sync.ned

Small sample network with three hosts and one switch that shows how to configure the TTEthernet Model. Configuration in omnetpp.ini

  • Uses 3ms Cycletime
  • 80ns Tick length

            datarate = 100Mbps;
        }

        unit1: Unit1 {
